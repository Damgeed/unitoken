How GlbTOKEN protects your data, keys, and payments

Security Score
🔑

API Key Authentication

Every request is authenticated with unique Bearer tokens. Keys are generated securely per user — no shared credentials, no password-based API access.

🔒

HTTPS Encryption

All API traffic is encrypted with TLS 1.3. Data in transit is fully protected — no cleartext communication, no unencrypted endpoints.

🎫

Token-Based Access Control

Each API key maps to a specific user account and token balance. No shared secrets, no hardcoded credentials. Revoke or rotate keys instantly from the dashboard.

⏱️

Rate Limiting per Key

Configurable per-key rate limits prevent abuse and ensure fair resource distribution. Limits are applied at the API gateway before reaching our infrastructure.

📊

Usage Monitoring & Alerts

Real-time dashboards track every API call, token spend, and model usage per key. Configure usage alerts to notify you before hitting balance thresholds.

💳

No Credit Card Storage

Payment data is handled exclusively by PCI-compliant partners — Stripe for card payments globally, Paystack for African markets (bank transfer, USSD, Mobile Money). GlbTOKEN never stores, processes, or touches raw card details. Crypto payments go directly on-chain.
